# Checkbox Module - POM Implementation

## Application Under Test
Website: https://the-internet.herokuapp.com/checkboxes  
Module: Checkboxes

---

## TC_01 - Verify first checkbox can be checked

**Precondition:**  
User is on checkbox page

**Test Steps:**
1. Navigate to checkbox page
2. Select first checkbox
3. Verify checkbox is selected

**Expected Result:**  
First checkbox should be checked.

---

## TC_02 - Verify second checkbox can be unchecked

**Precondition:**  
User is on checkbox page

**Test Steps:**
1. Navigate to checkbox page
2. Uncheck second checkbox
3. Verify checkbox is not selected

**Expected Result:**  
Second checkbox should be unchecked.
